تانى حاجة معانا النهارده ازاى نقدر نغير ال time و ال date

علشان تقدر تغير ال date و ال time كل اللى هاتحتاجه انك تختار Date and Time Settings من system settings او انك تضغط right click على ال time فى ال right panel من الشاشة عشان تقدر تغيره

Network Time Protocol هى طريقة مشهورة جدا فى نظام لينكس .. وهى ان ال time وال date هايتم تغيرهم عن طريق الانترنت و هو بيشوف المكان اللى انت فيه و بيقدر يعرف الوقت عندك كام .. بس طبعا لو عايز تغيره تقدر تغيره تانى زى ماتعلمنا

Section 2

هانتكلم هنا عن ال Network configuration

فى نظام لينكس ملفات عشان تساعدنى فى انى اغير ال network بتاع النظام بس الملفات ديه بتتغير من توزيعة للتانية .. و كمان التغيير فيها مش سهل

عشان كده فيه tool فى لينكس بتساعدنى انى اعمل customize لل network بطريقة سهلة وهى ال Network manager

ال Network manager هو المسئول انه يعرضلى ال networks المتاحة ليه سواء كانت wifi او wired او حتى كمان كانت mobile broadband

Wired and Wireless Connections

انك تعمل wired connection فى اغلب الاحيان مش بتحتاج انك تعمل configuration بايدك لان ال Network manager بيعمل ده automatic

طيب فيه حالات بيكون فيها الجهاز static على network معين و بيحتاج فيها انك تعرف ال wired network بنفسك و ده انت ممكن تعمله ببساطة بايدك عن طريق ال Network manager

لكن فى حالة ال wirless مش بيقدر يعمل connect من نفسه لازم تقدر تعرف ال wirless و تدخل ال password لو فيه تعرفه و ده برضة سهل انك تعمله بنفسك فى ال network manager

طيب ازاى نقدر باه نعرف wireless جديد فى centOS

  1. Right-click the Network Manager icon.
  2. اختار Enable wireless
  3. Click the Network Manager icon.
  4. اختار ال network اللى عايز تعمل connect معاها
  5. دخل ال password و لو صح هايتعمله save ومش هاتحتاج كل مره تدخله تانى
  6. عشان تعرف انت connect بانهى شبكة و لو عايز تعمل disconnect دوس على Network Manager icon

طريقة تعريف ال wireless فى ubuntu

  1. In top panel, click Network Manager.
  2. اختار Enable wireless
  3. اختار ال network اللى عايز تعمل connect معاها
  4. دخل ال password و لو صح هايتعمله save ومش هاتحتاج كل مره تدخله تانى
  5. عشان اعدل فى network قديم اضغط على Edit Connections.

فى opensuse كل ال wired و ال wireless و ال vpn تقدر تعملها add و edit من Network Connections

زى اللى فى المثال ده acces network in OpenSuse
